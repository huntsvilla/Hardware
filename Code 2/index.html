Input Device

Input Devices give information for the computer to process. An example of popular input device would be a plugging in a keyboard and having your computer process its data.

Output Device

Similar to an Input devices an output device doesnt give information it recieves it. For example from a compter you can give data to your printer you have connected, telling the printer what to print.

Computer Processing Unit

The CPU is in otherwords the brain of the computer. The CPU has the important job of processing information you give the computer.

Arithmetic logic unit

The ALU performes arithemetic operations for your computer. It also acted as a base design for the Central processing unit

Control Unit

The CU is a very important component of the CPU thats in charge of directing the operations of the processors.

Random Access Memory

Random access memory also known as RAM is a form of memory used to store active data like code or games. RAM is important because it can be read or changed however.

Optical Memory

Optical memory is like RAM but instead of being stored on your computer its stored on a DVD or disc of some sort. RAM is easier and faster to usen so I havent really used Optical memory that much.

Magnetic Memory

Magnetic memory is an early form of RAM. An early form of Magetic energy would be a floppy disk. Today you can see magnetic storage on credit cards.

Flash Memory

Flash memory does not need any power to store data. A USB drive is a type of flash memory that we use often today.

Logic Gate

Logic Gates are are a key building block of intergrated circuits. Their job is to perform basic functions.
